extend(itera)
-------------
The extend function extends the list by appending all the items from the iterable.
Also, terates over its argument and adding each element to the list and extending the list. The length of the list increases by number of elements in itâ€™s argument.

Syntax of my_list.extend(iterable) 
-------------
The syntax of the list.extend(iter) function is:

    my_list = ['geeks', 'for']
    another_list = [6, 0, 4, 1]
    my_list.extend(another_list)
    print my_list

Output:
['geeks', 'for', 6, 0, 4, 1]
